# 快速开始

本页适合第一次打开 GPT Image Playground 的用户。目标是完成配置，并生成第一张可继续编辑、可在历史里追踪的图片。

![工作台总览](./images/overview-workbench.png)

## 1. 先决定你用哪个模型供应商

应用内置这些供应商类型：

- **OpenAI Compatible**：OpenAI 官方接口，也包括兼容 OpenAI API 格式的第三方端点。
- **Google Gemini**：Gemini 图片模型。
- **Seedream**：豆包/火山方舟 Seedream 系列。
- **SenseNova**：SenseNova U1 Fast。

如果你只想使用 OpenAI 官方模型，通常只需要填写 OpenAI API Key，Base URL 可以留空。如果你使用第三方中转、自托管模型或多个账号，建议在系统设置里保存为命名端点。

## 2. 打开系统设置

点击右上角 **Settings**，进入 **供应商 API 配置**。

![供应商设置](./images/settings-providers.png)

你可以在这里完成：

- 给每个供应商填写 API Key 和 Base URL。
- 为同一供应商新增多个命名端点，比如“官方账号”“公司中转”“本地网关”。
- 限制某个端点可用的模型列表。
- 添加该端点提供的自定义模型 ID。
- 选择默认端点，后续高级选项会优先使用它。

配置会保存在当前浏览器中，立即生效，不需要重启服务。优先级是：**界面设置 > 环境变量 > 默认值**。

## 3. 输入提示词

回到主界面，在提示词框输入你想生成的画面。例如：

```text
生成一张适合新品发布会使用的科技产品主视觉，主体清晰，留出标题区域，质感高级。
```

你也可以点击 **提示词模板**，从模板库里选择一个更完整的起点。

## 4. 检查高级选项

点击提示词框下方的 **高级**，检查当前供应商、模型、尺寸、质量和输出格式。

![高级选项](./images/advanced-options.png)

常见选择建议：

- 想快速看方向：数量选 2-4 张，质量先用自动或中等。
- 想要最终可用图：质量选高，尺寸按使用场景选择横向、纵向、正方形或自定义。
- 想看生成过程：在支持的模型上开启流式预览。
- 使用 Seedream 或 SenseNova：优先使用界面里的一等控件，不必手写 JSON。

## 5. 生成并查看结果

点击 **开始生成**。生成中的任务会进入任务队列；完成后图片会显示在输出区，也会写入右侧历史。

生成结果可以继续做三件事：

- 点击图片放大全屏查看。
- 点击 **编辑**，把当前图片作为参考图继续修改。
- 在历史卡片中查看提示词、参数、耗时和费用估算。
