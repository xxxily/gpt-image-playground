# 工作台界面说明

主界面由四个核心区域组成：顶部应用栏、左侧输入区、中间输出区、右侧历史区。它们共同形成“输入需求 -> 生成结果 -> 继续编辑 -> 沉淀历史”的工作流。

![工作台总览](./images/overview-workbench.png)

## 顶部应用栏

顶部展示应用名称，并提供几个全局入口：

- **主题切换**：在浅色/深色主题之间切换。
- **关于**：查看版本、项目地址、作者信息和更新检查。
- **Settings**：进入系统配置，包括供应商、提示词润色、存储、云同步和桌面端设置。

如果后台配置了页面顶部横幅，应用栏下方会展示该展示位；未配置、接口失败或图片加载失败时不会占用核心操作区域。顶部横幅的展示比例由对应展示组配置，PC 和移动端保持同一比例；具体说明见 [展示内容与后台管理使用手册](./展示内容与后台管理使用手册.md#11-展示组比例)。

## 输入区

输入区默认是生成模式。只要添加源图片，就会自动变成编辑模式。

如果部署者配置了头部展示位，输入区标题右侧会显示 `4:1` 展示横幅；未配置时界面保持原样。

主要操作：

- 在提示词框描述你想要的图片。
- 点击 **提示词模板** 打开模板库。
- 点击 **润色**，让润色模型改写当前提示词。
- 点击 **分享**，生成可复用的配置链接；管理员提供分享展示 `Profile ID` 时，可以给链接关联公开的 `promoProfileId`。
- 点击 **模板** 或直接输入 `/`，快速搜索模板。
- 点击 **历史**，找回最近用过的提示词。
- 点击 **高级**，配置供应商、模型和图片参数。
- 使用自定义分辨率时，可以先输入想要的宽高；如果不满足模型限制，界面会按当前比例推荐一个可提交尺寸并支持一键套用。
- 上传、拖入或粘贴源图片，进入编辑图片流程。

常用快捷键：

- `Ctrl/⌘ + Enter`：提交生成或编辑任务。
- `/`：在提示词中唤起模板搜索。
- 全屏预览时 `Esc` 退出，左右方向键切换多图。

## 输出区

输出区显示当前任务的结果。

- 生成中会显示等待状态、耗时或流式预览。
- 多图结果支持网格视图和单图视图。
- 点击图片进入全屏预览；预览左上角会短暂显示图片尺寸和文件大小，无法读取的信息显示为未知。
- 全屏预览的图片信息和底部缩放控制会在静置后自动淡出，桌面端移动鼠标、移动端轻点图片会再次显示。
- 单图视图下可以直接发送到编辑区。

## 历史区

历史区记录每一次生成或编辑。

![历史全屏预览](./images/history-preview.png)

如果后台配置了历史区横幅，历史列表上方会展示该展示位；展示内容异常时历史检索、预览和下载不受影响。历史横幅的展示比例由对应展示组配置，PC 和移动端保持同一比例。

历史卡片会显示：

- 生成或编辑标识。
- 模型、质量、格式、背景、审核参数。
- 耗时和图片数量。
- 提示词入口。
- 下载入口。
- 费用估算入口。

历史为空时，应用会展示内置项目信息图，帮助你快速理解一图速览、工作台总览、提示词资产、供应商管理、历史资产和分享同步等能力。示例不会写入你的真实历史。

## 全局图片输入

编辑图片有三种常用入口：

- 点击 **添加原图** 选择文件。
- 从桌面拖拽图片到页面。
- 在页面任意位置粘贴剪贴板中的图片。
- 在源图区点击 **从剪贴板添加图片**，由一次明确的用户操作读取系统剪贴板图片；浏览器不允许读取时，仍可聚焦源图区后使用粘贴快捷键。

上传源图片后，按钮会变成 **开始编辑**。

多张源图片会按源图区缩略图顺序提交给后续接口；如果上传、拖入或粘贴顺序不对，可以直接拖动缩略图重新排列。

## 创作工作空间与创作资源分区

右下角全局功能菜单平铺提供 **工作空间**、**资源物料仓库** 和 **素材灵感库**。工作空间用于按主题、项目或客户隔离创作历史；资源物料仓库和素材灵感库仍是全局可复用的素材入口。桌面端鼠标移动到悬浮按钮会直接展开功能菜单，移动端仍可点击按钮打开。鼠标移动到一级菜单项时，会在旁边展开左侧打开、右侧打开、抽屉打开等二级操作，不需要进入子菜单后再返回。二级菜单会在右侧空间足够时优先向右展开，空间不足时再切到左侧。

在桌面宽屏下，这些面板可以停靠在左侧或右侧。工作空间和素材面板停靠在同一侧时，会在同一个分区内用标签切换；停靠在不同侧时，可以左右同时显示，主工作台保持在中间。移动端和窄窗口会自动使用全屏 Drawer。

分区面板支持：

- 拖动主工作台与左侧或右侧面板之间的分隔线调整宽度。
- 使用键盘方向键调整分隔线，双击恢复默认宽度。
- 收起、展开或关闭分区面板；收起后可点击图标或竖排标题区域展开，再次打开会恢复本机宽度和停靠侧偏好。
- 在工作空间面板中新建、进入、置顶、重命名、调整颜色标识、归档或删除工作空间。工作空间按置顶优先、创建时间倒序显示，进入工作空间不会改变列表顺序。删除用户创建的工作空间时，需要在项目内确认弹窗中输入准确名称。
- 在资源物料仓库里发送图片素材到编辑区，面板会保持打开，便于继续选择素材。
- 在素材灵感库里打开站点；站点列表会先显示名称首字，favicon 加载成功后替换为站点图标。默认站点优先提供花瓣、堆糖、站酷等国内设计社区，同时保留 Pinterest、Dribbble、Behance 等知名海外设计站点，不默认提供以摄影图库为主的站点。内置站点默认外部打开，避免第三方 iframe 限制导致空白或报错；在“管理网站”里可以为自有素材站点切换为站内 iframe 打开。

主工作台参数概要下方、源图区上方会显示当前工作空间入口。点击这个入口会直接打开工作空间面板；进入某个工作空间后，新提交的生成、编辑、图生文和视频任务都会写入该空间；切换工作空间不会改变已经排队或运行中任务的归属。默认工作空间用于兼容旧历史，可以重命名和清空内容，但不能直接删除。

分屏不会读取第三方 iframe 内部内容。用户从灵感网站复制图片后，需要回到主工作台源图区粘贴，或点击源图区的剪贴板读取按钮。

## 任务队列

系统支持多个任务排队执行。并发数量可以在 **Settings -> 运行与存储** 中调整。

任务状态包括：

- 排队中。
- 处理中。
- 流式生成。
- 完成。
- 出错。
- 已取消。

生成中可以取消任务，失败任务可以关闭或重试。重试会使用该失败任务提交时的提示词、源图片和参数快照，不会读取当前输入区已经改动过的内容；关闭或清理任务后，对应快照会释放。

## 桌面端展示内容读取

桌面端可以在 **Settings -> 桌面端设置** 中选择展示内容接口来源：

- `当前站点`：默认读取当前应用的 `/api/promo/placements`。
- `自定义域名`：填写服务域名，系统自动拼接读取接口。
- `完整接口`：直接填写完整读取接口。
- `关闭`：不请求展示内容接口，展示位保持隐藏。

展示内容请求失败、离线或超时时不会影响生成、编辑、历史和分享流程。
