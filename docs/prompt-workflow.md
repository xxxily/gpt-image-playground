# 提示词工作流

提示词不是一次性输入框，而是可以被搜索、保存、复用、润色和分享的资产。GPT Image Playground 把提示词相关能力集中在输入框上方和旁边。

![提示词模板库](./images/prompt-template-library.png)

## 提示词模板库

点击 **提示词模板** 可以打开模板库。模板库包含多个创作场景：

- 风格转换。
- 电商商品图。
- 社交媒体内容。
- 品牌营销。
- 餐饮美食。
- 时尚美妆。
- 地产空间。
- 教育培训。
- 游戏概念。
- 科技产品。
- 旅行文旅。
- 健康生活。
- 头像人像。
- 商务办公。
- 节日季节。
- 纹理背景。

你可以按分类浏览，也可以搜索名称、分类和提示词内容。

## 快速搜索模板

在提示词框里输入 `/`，会出现模板搜索列表。也可以点击工具栏里的 **模板** 按钮插入 `/`。

适合场景：

- 写到一半时插入完整模板。
- 根据关键词快速找“商品主图”“小红书封面”“职业头像”等模板。
- 用键盘上下选择，按 Enter 应用。

## 保存自己的模板

模板库支持保存本地模板。常见做法：

1. 在提示词框写出一条你满意的提示词。
2. 打开 **提示词模板**。
3. 点击 **添加模板**。
4. 填写模板名称和分类。
5. 保存后，它会出现在“我的模板”或对应分类里。

默认模板不能直接修改，但可以复制成一份本地模板后再编辑。

## 管理本地模板

在模板库的 **管理本地** 中，可以：

- 编辑本地模板。
- 删除本地模板。
- 导出模板 JSON。
- 导入模板 JSON。
- 置顶常用分类。

模板保存在当前浏览器中。如果你换设备，需要手动导入，或使用云同步恢复。

## 提示词历史

每次提交生成或编辑后，提示词会进入历史。点击工具栏里的 **历史** 可以搜索最近用过的提示词。

适合场景：

- 找回刚刚试过的版本。
- 把历史提示词作为新一轮修改起点。
- 清理不再需要的历史。

历史数量可以在 **Settings -> 运行与存储 -> 提示词历史数量** 中调整。

## 提示词快捷按钮

输入框下方的快捷按钮右侧优先显示，宽度不足时默认靠右展示最近侧的操作。按钮显示可以在 **Settings -> 常规 -> 提示词按钮** 中调整；视频入口当前默认隐藏，功能调好前需要手动开启。

批量入口在图生文和视频模式下会保留在工具栏中，但处于禁用状态。通过分享链接临时使用 API Key 且分享者勾选 **禁止批量能力** 时，接收者同样只能看到禁用状态，不能创建批量任务。

批量预览不会在刷新页面后自动回到右侧结果区。未提交的批量草稿仍会保存在本地；重新打开 **批量** 面板时，可以在恢复条中手动恢复或丢弃。

## 一键润色

输入提示词后，点击 **润色** 可以让润色模型重写当前提示词。

润色支持：

- 默认内置润色预设。
- 多个内置预设。
- 你自己保存的润色提示词。
- 临时自定义润色要求。

润色模型选择、批量规划模型选择、思考模式和预设顺序都在 **Settings -> 提示词润色配置** 中设置。润色与批量规划不再有自动默认模型；用户需要先选择 OpenAI/OpenAI 兼容或 Anthropic/Anthropic 兼容端点，再点击模型状态框弹窗读取该端点最新模型列表并单选绑定。如果还没有这类端点，先到供应商端点管理添加端点。

## 分享提示词和配置

点击 **分享** 可以把当前提示词、模型、供应商端点和可选 API 配置写入链接。

![分享链接](./images/share-dialog.png)

默认推荐只分享提示词、模型和端点信息。API Key 默认不包含，只有你明确勾选并确认风险后才会写入链接。

需要保护参数时，可以启用密码加密分享。

启用密码加密后，解密密码和确认密码完全一致时，面板会自动把“生成的分享链接”更新为带 `sdata` 的加密链接；创建短链会使用这条当前链接。
