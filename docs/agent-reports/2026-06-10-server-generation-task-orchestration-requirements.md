## 后续建议

- 后续若要让 Tauri desktop 新提交也走 managed-task，需要先设计原生任务服务配置、凭证委托、HTTP client 和静态导出访问路径，不能直接依赖当前 Web API Route。
- Phase 7 的 S3、视频、批量、图生文、多实例和多租户仍不属于 P0；需要单独立项后再扩展，不能混入 Phase 0-6 的 P0 验收口径。
- 真实 provider adapter 接入后，必须复测供应商 429/5xx 分类、幂等保护、重复付费风险、资产 manifest 脱敏和持久审计。
