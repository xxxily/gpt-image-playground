# 更新日志

## v2.0.0 - 2026-05-01

### 重点更新

- 新增 Tauri 桌面应用支持，发布流程可为 macOS、Windows 和 Linux 自动构建安装包。
- 新增 Gemini 图像 Provider，并引入 Provider-aware 模型注册与任务执行架构，为后续多模型扩展打基础。
- 新增提示词模板与斜杠命令系统，可通过 `/` 快速插入常用图片生成/编辑模板，并支持键盘导航。
- 新增并发任务系统，支持任务持久化、任务卡片、任务追踪器和毫秒级耗时展示。
- 新增 UI 系统设置面板，支持 API Key、Base URL、图片存储模式、连接模式和自定义模型配置热更新。

### 交互体验

- 历史图片支持双击全屏预览，并可从历史记录或预览中快速“发送到编辑”。
- 图片查看器支持点击缩放、滚轮缩放、双指缩放、拖拽移动、点击外部关闭和 ESC 退出。
- 编辑模式支持全局剪贴板粘贴和全局拖拽上传，图片输入路径更顺手。
- 设置页、主工作区、历史面板和辅助对话框完成移动端与亮色/暗色主题适配。

### 稳定性与错误处理

- 强化 API 错误处理与连接策略，新增环境变量示例，第三方中转/直连场景提示更清晰。
- 修复取消任务时缺失 AbortController、设置保存/重置后的状态同步、页面缩放误触和右键拖拽误触等问题。
- 修复缩放查看器拉伸变形、点击外部关闭、垂直居中和布局标签换行等 UI 问题。

### 国际化与文档

- 完成整体 UI 简体中文翻译，并修复中文字符串引号问题。
- README 补充连接模式、系统设置和构建说明。

### 构建与发布

- 新增跨平台 Tauri 构建工作流，并在 v2.0.0 中升级为 tag 驱动发布：推送 `v*` 标签后会校验版本、生成 GitHub Release 描述并上传桌面安装包。
- 构建流程新增版本一致性检查、Lint、TypeScript 类型检查、Web 构建预检和工作流产物上传，减少发布失败和描述过简的问题。

### 升级注意事项

- 生成/编辑表单已统一为新的工作流，历史使用习惯可能略有变化。
- 图像任务统一经过 Provider-aware 执行层，建议同时验证 OpenAI 与 Gemini 配置。
- 新增任务持久化与主题系统后，如遇到旧配置显示异常，可在系统设置中重置相关配置。
