# 更新日志

## v2.3.0 - 2026-05-04

### 重点更新

- 全屏图片预览升级为跨图片的相册式查看体验，生成结果、历史记录和输出区多图都能在同一预览中左右滑动、拖拽或使用方向键切换。
- 新增可取消的图片切换过渡：拖拽不足阈值会平滑回弹，持续越过阈值才切换到相邻图片，减少误触切图。
- 优化移动端缩放计算，在小屏幕上按视口和图片尺寸动态限制初始缩放，打开大图时更容易完整浏览。

### 交互体验

- 抽象 ZoomViewer gallery 手势辅助逻辑，放大后优先拖拽查看图片细节，横向越过边界后再触发跨图切换。
- 历史面板、图片输出区和编辑表单统一接入多图预览上下文，发送到编辑、历史查看和当前生成结果之间的预览行为更一致。

### 构建与部署

- Dockerfile 与 docker-compose 支持构建时传入运行环境变量，便于在容器镜像构建阶段注入生产配置并保持部署脚本稳定。
- README 补充相册式切换交互说明，便于用户了解多图预览、拖拽过渡和键盘切换行为。

### 稳定性与测试

- 新增 ZoomViewer gallery 手势与缩放逻辑测试，覆盖拖拽阈值、可取消切换、边界判定和移动端缩放计算等关键路径。
- 同步更新 Web 与桌面端版本号到 `2.3.0`，确保 tag 驱动的 GitHub Release 和桌面端构建流程通过版本一致性校验。

## v2.2.2 - 2026-05-03

### 重点更新

- 新增 Umami 统计分析配置，部署时可通过 `UMAMI_SCRIPT_URL` 和 `UMAMI_WEBSITE_ID` 注入统计脚本；未完整配置时不会加载，避免误发请求。
- 全屏图片预览支持手机相册式多图切换：左右拖拽时会先露出相邻图片，松手距离不足会平滑回弹取消切换，持续拖过阈值才缓动切到上、下一张；放大后仍会先拖拽查看细节，越过水平边界后再触发切换，并支持键盘左右方向键。
- 提示词润色链路增强了异常解析与结果归一化，兼容更多 OpenAI-compatible 响应格式，并在失败时给出更清晰的提示。
- 安全分享链接支持在复制时附带内联解密密码，接收者打开完整链接即可自动解密，同时仍会清理地址栏中的敏感片段。

### 配置与分享

- 共享配置在直连模式下新增专用辅助逻辑，临时分享配置会在运行时生效，减少保存到本地设置前后的状态不一致。
- 简化安全分享密码的校验规则，允许更轻量的分享密码，降低临时协作场景的使用门槛。
- 设置面板对敏感字段的浏览器自动填充做了隔离处理，减少 API Key 或密码被误填到不相关输入框的可能性。

### 稳定性与测试

- 补充 API 错误处理、提示词润色、分享加密、URL 参数和共享配置相关测试，覆盖第三方接口异常、直连配置和加密分享的关键边界情况。
- 同步更新 Web 与桌面端版本号到 `2.2.2`，确保 tag 驱动的 GitHub Release 和桌面端构建流程通过版本一致性校验。

## v2.2.0 - 2026-05-03

### 重点更新

- 新增安全分享链接能力，可选择分享提示词、模型、API Base URL、自动生成开关等配置，并支持使用密码加密整段分享数据。
- 新增接收者配置保存选择，打开分享链接时可选择仅本次使用、保存到本地设置或忽略分享中的 API 配置。
- 新增提示词润色功能，可在输入区一键优化提示词，并可独立配置润色模型的 Base URL、API Key、模型 ID 和系统提示词。

### 交互体验

- 清空历史记录改为二次确认弹窗，并在清空前做数据归一化处理，降低误操作和旧数据结构带来的风险。
- 关于信息增加 GitHub 入口，网址改为生产站点 `img-playground.anzz.site`，展示文本更短，便于在一行内阅读。
- 编辑表单移除额外的 overscroll 限制，长内容滚动时更贴近页面整体滚动行为。

### 稳定性与发布

- 增加 URL 参数、加密分享、服务端 URL 安全校验、提示词润色和历史清理相关测试，覆盖分享与配置链路的关键边界情况。
- Docker 构建上下文排除本地产物目录，减少部署上传体积和误带本地缓存的可能性。
- 同步更新 Web 与桌面端版本号到 `2.2.0`，确保 tag 驱动的桌面构建发布流程能够通过版本一致性校验。

## v2.1.0 - 2026-05-02

### 重点更新

- 新增图片代理接口，支持服务端拉取跨域图片并转入应用存储链路，解决历史图片 URL 在编辑、保存和再次使用时的跨域限制。
- 重构图片保存逻辑，生成结果可携带图片 URL 并写入历史记录，为远程图片、代理图片和本地存储的统一处理打基础。
- 新增滚动感知的右上角控件显示策略，向下浏览内容时自动收起操作按钮，回到顶部后恢复显示，减少画布区域遮挡。
- 排队中的图片任务现在也可以取消，任务尚未开始执行时会被直接移出队列，避免无效请求继续占用任务队列。

### 交互体验

- 设置面板抽象统一的 ProviderSection 区块，OpenAI、Gemini 和自定义配置的呈现方式更一致。
- 调整提示词模板和顶部标题文案，首页信息层级更清晰，常用入口更容易识别。
- 优化历史面板、任务卡片和任务追踪器细节，使任务状态、图片 URL 和取消操作的反馈更连贯。

### 稳定性与兼容性

- 强化图片代理和下载失败时的错误类型与提示，接口会返回更可读的失败原因。
- 更新 API 图片生成流程，让文件系统存储、IndexedDB 存储和 URL 化图片路径更稳定地协同工作。
- 补充历史记录类型字段，兼容图片 URL 与已有 base64、本地路径历史数据。

### 升级注意事项

- 本版本更新了历史图片的存储字段，发布后建议验证“生成图片 → 历史查看 → 发送到编辑”的完整链路。
- 图片代理会从服务端拉取外部图片，请确认生产环境允许访问目标图片域名，并关注服务器带宽与超时策略。

## v2.0.0 - 2026-05-01

### 重点更新

- 新增 Tauri 桌面应用支持，发布流程可为 macOS、Windows 和 Linux 自动构建安装包。
- 新增 Gemini 图像 Provider，并引入 Provider-aware 模型注册与任务执行架构，为后续多模型扩展打基础。
- 新增提示词模板与斜杠命令系统，可通过 `/` 快速插入常用图片生成/编辑模板，并支持键盘导航。
- 新增并发任务系统，支持任务持久化、任务卡片、任务追踪器和毫秒级耗时展示。
- 新增 UI 系统设置面板，支持 API Key、Base URL、图片存储模式、连接模式和自定义模型配置热更新。

### 交互体验

- 历史图片支持双击全屏预览，并可从历史记录或预览中快速“发送到编辑”。
- 图片查看器支持点击缩放、滚轮缩放、双指缩放、拖拽移动、点击外部关闭和 ESC 退出。
- 编辑模式支持全局剪贴板粘贴和全局拖拽上传，图片输入路径更顺手。
- 设置页、主工作区、历史面板和辅助对话框完成移动端与亮色/暗色主题适配。

### 稳定性与错误处理

- 强化 API 错误处理与连接策略，新增环境变量示例，第三方中转/直连场景提示更清晰。
- 修复取消任务时缺失 AbortController、设置保存/重置后的状态同步、页面缩放误触和右键拖拽误触等问题。
- 修复缩放查看器拉伸变形、点击外部关闭、垂直居中和布局标签换行等 UI 问题。

### 国际化与文档

- 完成整体 UI 简体中文翻译，并修复中文字符串引号问题。
- README 补充连接模式、系统设置和构建说明。

### 构建与发布

- 新增跨平台 Tauri 构建工作流，并在 v2.0.0 中升级为 tag 驱动发布：推送 `v*` 标签后会校验版本、生成 GitHub Release 描述并上传桌面安装包。
- 构建流程新增版本一致性检查、Lint、TypeScript 类型检查、Web 构建预检和工作流产物上传，减少发布失败和描述过简的问题。

### 升级注意事项

- 生成/编辑表单已统一为新的工作流，历史使用习惯可能略有变化。
- 图像任务统一经过 Provider-aware 执行层，建议同时验证 OpenAI 与 Gemini 配置。
- 新增任务持久化与主题系统后，如遇到旧配置显示异常，可在系统设置中重置相关配置。
